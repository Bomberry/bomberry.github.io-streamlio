Fonts

The styling of the Streamlio website uses text, colors, gradients, and images from the branding guideline to keep a consistant visual message for the website.

Main Font - Cambay - Sans Font

The main font can be applied using .font-sans class.

The primary font is Cambay, and is a sans-serif font. It is used for the main content of the website.

Alternate Font - Roboto - Sans Alt

The alternate font can be applied using .font-sans-alt class.

The alternate primary font is Roboto, and is also a sans-serif font. It is used only in special cases.

Alternate Font Condenced - Roboto Condensed - Sans Condenced

The alternate condenced font can be applied using .font-sans-condensed class.

The alternate condenced font is Roboto Condenced, and is also a sans-serif font. It is used only in special cases.

Secondary Font - Martel - Serif Font

The secondary font can be applied using .font-serif class.

The secondary font is Martel, and is a serif font. It is used only in cases where personal emotion is expressed, such as a testimonial, or in a blockquote (tag/element). It is also used for figure captions.

FontAwesome

FontAwesome is used for icongraphs used throughtout the site.

Buttons

Buttons can be declared as either BUTTON tag, or as an A tag with .button class applied to it.

Buttons can be colored by adding .button-COLOR_NAME class to the element.

Buttons can be disabled by adding .button-disabled class to the element.

Note: The BUTTON tag has a set font size applied. Applying .button class on a text element (link, paragraph, heading, etc...) the element inherits the font-size from the parent.

Button Tag

Standard button tag elemnts.

<button class="button-default">Default</button>
<button class="button-default-reverse">Default Reverse</button>
<button class="button-default-reverse button-disabled">Default Reverse Disabled</button>
<button class="button-apple">Button Apple</button>
<button class="button-apple-reverse">Button Apple Reverse</button>
<button class="button-apple button-disabled">Button Apple Disabled</button>

Button Sizes

Both the button tag and .button classe have additional larger sizes as follows.

<button class="button-default">Default Button</button>
<button class="button-default-reverse button-large">Button Large</button>
<button class="button-apple button-larger">Button Larger</button>
<button class="button-apple-reverse button-largest">Button Largest</button>
<button class="button-indiglow button-jumbo">Button Jumbo</button>
